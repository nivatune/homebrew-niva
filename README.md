# Niva Homebrew Tap

Homebrew formulae for [Niva](https://github.com/nivatune/user), a text-based
music engraving tool for jazz and pop lead sheets.

## Install

```sh
brew tap nivatune/niva
brew install niva
```

## Upgrade

```sh
brew update
brew upgrade niva
```

The `niva` formula is updated automatically on each Niva release; you do not
edit it by hand.
